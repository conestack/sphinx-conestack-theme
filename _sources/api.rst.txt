API documentation
=================

API Documentation can be generated with
`sphinx.ext.autodoc <https://www.sphinx-doc.org/en/master/usage/extensions/autodoc.html>`_.

For example, the following directive::

    Python Datetime
    ---------------

    .. automodule:: datetime
       :members:
       :undoc-members:
       :show-inheritance:

produces the following output:

Python Datetime
---------------

.. automodule:: datetime
   :members:
   :undoc-members:
   :show-inheritance:
